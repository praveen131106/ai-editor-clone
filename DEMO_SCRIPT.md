# NovaGlow AI - Contest Demo Script

## Overview
3-minute walkthrough for judges demonstrating key features and capabilities.

---

## Demo Setup
- **Device**: Use a physical device or high-quality simulator
- **Test Media**: Pre-load with a high-quality portrait photo and a short video clip
- **Environment**: Good lighting, stable internet connection
- **Time**: 3 minutes maximum

---

## Demo Tips

- Clear app cache
- Close other apps
- Test all features beforehand
- Have test media ready
- Ensure device is charged

### During Demo
- Speak clearly and confidently
- Keep pace steady (don't rush)
- Highlight key features
- Show smooth transitions
- Handle any errors gracefully
- Stay within 3-minute limit

### After Demo
- Be prepared for questions
- Know technical details
- Explain architecture briefly
- Mention future roadmap
- Thank judges

## Common Questions & Answers

**Q: Does this require a backend?**
A: No, all demonstrated features work locally. Only advanced AI features like Avatar Generator would require backend configuration.

**Q: What platforms does it support?**
A: iOS and Android via React Native / Expo.

**Q: How does the object eraser work?**
A: It uses a smart blur effect to visually remove objects. For production, this would use AI-powered inpainting.

**Q: Can users save their work?**
A: Yes, edited photos and videos can be exported to the device gallery.

**Q: What's the tech stack?**
A: React Native, Expo, TypeScript, with local image processing.

**Q: Are there any limitations?**
A: Advanced AI features require backend. Video editing is basic (trimming, FX, text). No cloud storage integration yet.

## Backup Plan

If something goes wrong during demo:

1. **App crashes**: Relaunch quickly, continue with next feature
2. **Feature fails**: Skip to next feature, mention it's a known issue
3. **Media won't load**: Use stock media instead
4. **Export fails**: Mention export feature, move to conclusion
5. **Time running out**: Skip to conclusion, summarize features

---
